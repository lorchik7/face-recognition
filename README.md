## About The Project

This [React.js](https://reactjs.org/) app detects human faces on given photo.

<!-- GETTING STARTED -->
## Getting Started
### Prerequisites

You need to have Node JS installed.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lorchik7/face-recognition.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
